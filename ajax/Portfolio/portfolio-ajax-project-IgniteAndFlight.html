Ignite and Flight

Project Info:

Created for the Falmouth university Pheonix Jam you play as a pheonix flying around floating islands. Fly close to build up points or dive into flame to be reborn in a new world. For this project I was responsible for everything apart from 2D art.

Project Details:

  • Industry:Videogames
  • Technologies:C#, Unity, Blender
